# 🚀 AI Career Guidance System

An AI-powered Career Guidance System that helps students identify suitable career paths based on their skills, interests, and academic strengths. The system analyzes user inputs and provides personalized career recommendations along with skill-gap insights.

## 📌 Project Overview

Choosing the right career path can be challenging for students. This project uses Artificial Intelligence and Data Analysis techniques to recommend career options based on a user's profile.

The application evaluates skills, interests, and educational background to suggest careers such as:

- Data Scientist
- Machine Learning Engineer
- AI Engineer
- Data Analyst
- Software Developer
- Business Analyst
- Cybersecurity Analyst

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/AI-Career-Guidance-System.git
```

Navigate to the project directory:

```bash
cd AI-Career-Guidance-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

## 📊 How It Works

1. User enters skills and interests.
2. System processes the input data.
3. Career matching algorithm calculates scores.
4. Suitable career paths are recommended.
5. Visual charts display career compatibility.
